Bagging (Bootstrap Aggregation)

模型融合技术应用广泛，可以通过弱学习器得到强学习器，决定融合效果的关键因素在于，这些弱学习器要具有多样性，这样才能组合出更强大和全面的学习器。增加多样性的一些常见手段如下：

  • 完全不同模型
  • 同样的模型，但是参数不同
  • 算法本身具有随机性：例如PLA算法
  • 数据具有随机性

针对上面的第一个和第四个手段，产生了两种对应的模型融合的主要思路：blending和bagging。

前面所讲的这些模型融合方法都属于blending，特点是选择不同的模型，在每个模型单独学习完之后，再进行融合，也就是给不同的模型不同的权重。

而bagging的特点是，利用数据有放回的随机采样，对训练数据进行抽样，然后在不同的数据集上训练相同的模型，最后的结果由这些模型投票产生。bagging是一种元算法。
