机器学习应用概述

发表于 2016-04-04   |   分类于 machine learning   |  

如今，机器学习技术已经慢慢普及，在生活中的很多地方都能看到机器学习的应用。本文就来简单盘点一下，机器学习有哪些主要的应用领域。

数据挖掘

提到机器学习的应用，最直接的就是数据挖掘了，数据挖掘，又叫数据库知识发现，是指从大量的数据或者数据库中提取有用信息。在数据挖掘中，大量利用了机器学习算法，例如各种分类算法。因此很多时候，很难将数据挖掘和机器学习完全分离开，但是我觉得，数据挖掘更强调的是解决实际的问题，因此会更关注大规模的数据。而机器学习一般不关注问题的规模，而是注重算法的有效性。而且机器学习主要关注的是监督学习，而数据挖掘则主要关注无监督学习。

数据挖掘除了利用机器学习的算法外，还有一些特有的方法，例如关联分析。关联分析主要用来分析购物篮数据中的频繁模式和联系。主要的关联分析方法有Apriori和FP-growth算法。

自然语言处理

机器学习的另一主要应用就是自然语言处理了，自然语言处理（英语：Natural Language Processing，简称NLP）是人工智能和语言学领域的分支学科。在这此领域中探讨如何处理及运用自然语言；自然语言认知则是指让电脑“懂”人类的语言。

自然语言生成系统把计算机数据转化为自然语言。自然语言理解系统把自然语言转化为计算机程序更易于处理的形式。自然语言处理主要设计到两种数据类型：文本和语音。

针对文本数据，最常见的应用有文本分类、文本聚类。针对语音，有语音识别、语音合成等。而且世界上有很多种语言和文字，每种都有其特殊性，例如对于中文，有中文分词、词性标注、句法分析等等。还有用于不同语种的机器翻译等等。

搜索引擎

搜索引擎指自动从因特网搜集信息，经过一定整理以后，提供给用户进行查询的系统。搜索引擎大致可分为三个过程：

  • 搜集信息：利用网络爬虫，自动爬取互联网上的信息。
  • 整理信息：对搜集的信息进行预处理，这是搜索引擎最主要的部分，主要包括以下步骤：
    • 关键词提取
    • 网页排序
    • 倒排索引
  • 查询处理

其中，最关键的步骤在于网页排序，按照使用的方法的不同，可将搜索引擎分为3代：

  • 第一代技术，将互联网网页看作文本，主要采用传统信息检索的方法。
  • 第二代技术，利用互联网的超文本结构，有效地计算网页的相关度与重要度，代表的算法有 PageRank 等。
  • 第三代技术，有效利用日志数据与统计学习方法，使网页相关度与重要度计算的精度有了进一步的提升，代表的方法包括排序学习（learning to rank, L2R)、网页重要度学习、匹配学习、话题模型学习、查询语句转化学习。

推荐系统

推荐系统是对搜索引擎的延伸，主要是用来对用户进行个性化的推荐服务。在推荐系统中，主要包括两类推荐算法，一是基于内容的推荐，二是协同过滤。其中基于内容的推荐大量利用了机器学习中的算法。

而且在实际的推荐系统中，为了进一步提高推荐的全面性，融合各种推荐算法的结果，然后进行重排序。而重排序的过程主要利用了机器学习中的Learning to Rank的方法。
